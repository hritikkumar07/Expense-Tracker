# 💰 Expense Tracker

A full-stack Expense Tracker application that helps users manage their personal finances efficiently. The platform allows users to track income and expenses, monitor spending habits, and gain valuable financial insights through interactive dashboards and reports.

## ✨ Features

* ➕ Add, edit, and delete income & expense transactions
* 🏷️ Category-based financial tracking
* 📊 Interactive finance dashboard
* 📅 Daily, weekly, and monthly financial reports
* 📈 Data visualization with charts and analytics
* 🔒 Secure RESTful APIs
* 🎨 Responsive and modern UI built with Tailwind CSS
* ⚡ Scalable full-stack architecture

## 🛠️ Tech Stack

### Frontend

* ⚛️ React
* 🎨 Tailwind CSS

### Backend

* 🟢 Node.js
* 🚀 Express.js

### Database

* 🍃 MongoDB

### Tools
- 📮 Postman (API Testing)
- 📝 Git & GitHub

This project was built to provide a simple and intuitive way to manage personal finances while showcasing full-stack development skills, REST API integration, database management, and data visualization techniques.
